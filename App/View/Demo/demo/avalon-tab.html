使用avalon实现tab选项卡(源码)

页面代码

  1.  <!-- Nav tabs -->
  2.       <ul class="nav nav-tabs" role="tablist">
  3.         <li role="presentation" class="" ms-class='active:currentIndex===1' ms-click='tabClick(1);'><a href="javascript:void(0);" aria-controls="home" role="tab" >选项卡一</a></li>
  4.         <li role="presentation" ms-class='active:currentIndex===2' ms-click='tabClick(2);'><a href="javascript:void(0);" aria-controls="profile" role="tab" >选项卡二</a></li>
  5.         <li role="presentation" ms-class='active:currentIndex===3' ms-click='tabClick(3);'><a href="javascript:void(0);" aria-controls="messages" role="tab" >选项卡三</a></li>
  6.         <li role="presentation" ms-class='active:currentIndex===4' ms-click='tabClick(4);'><a href="javascript:void(0);" aria-controls="settings" role="tab" >选项卡四</a></li>
  7.       </ul>
  9.       <!-- Tab panes -->
  10.       <div class="tab-content">
  11.         <div class="tab-pane" ms-visible='currentIndex===1'>选项卡一内容<br/>111111</div>
  12.         <div class="tab-pane" ms-visible='currentIndex===2' >选项卡二内容<br/>222222</div>
  13.         <div class="tab-pane" ms-visible='currentIndex===3' >选项卡三内容<br/>33333</div>
  14.         <div class="tab-pane" ms-visible='currentIndex===4' >选项卡四内容<br/>44444</div>
  15.       </div>

avalon脚本代码

  1. <script type="text/javascript">
  2.     var model = avalon.define({
  3.         $id:'tab',
  4.         currentIndex:1,
  5.         tabClick:function(index){
  6.             model.currentIndex = index;
  7.         }
  8.     });
  9. </script>
